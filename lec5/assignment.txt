1. Difference between Name , * and Default Export.

Named exports are useful when you want to export multiple values from a module.
Default exports are useful when you want to export a single value.
Wildcard exports/imports are useful for importing multiple named exports under a single namespace.
import * as moduleExports from './moduleFile';
